# CurrencyConvert
使用C#和WPF完成的货币汇率转化小应用，使用网上的api获取相关汇率，存入sql server中，在复选框中选择货币后，提取相关数据进行转换。

增加了对C#和WPF的了解，增加了使用C#对sql增删改查操作的相关知识
